# CLDF dataset derived from Kim et al's "The Tripura of Bangladesh: A Sociolinguistic Survey" from 2011

## How to cite

If you use these data please cite
- the original source
  > Kim, Amy and Roy, Palash and Sangma, Mridul and Kim, Seung (2011) The Tripura of Bangladesh: A Sociolinguistic Survey. Journal of Language Survey Reports. Dallas: SIL International
- the derived dataset using the DOI of the [particular released version](../../releases/) you were using
